# Hydrosense
Repositório do projeto Hydrosense.

Hydrosense é um sistema inteligente de detecção de vazamentos de água e gás utilizando ESP32 e sensores piezoelétricos. A solução capta vibrações e padrões acústicos na tubulação para identificar vazamentos de forma precisa e não invasiva.
Sendo também, capaz de estimar a localização do problema ao longo da tubulação, facilitando a manutenção e reduzindo custos com inspeção.
Os dados são processados em tempo real e enviados para um aplicativo, onde o usuário recebe alertas imediatos e pode acompanhar o consumo, identificando possíveis desperdícios.
